# App de Finanças em Python

É um sistema que ajuda a ter um controle financeiro, permitindo registrar entradas e saídas, visualizar o extrato, calcular o saldo atual e exportar os dados para relatórios

# O que faz

- **Salva os dados**: Guarda tudo em um arquivo de forma automática.
- **Entradas e saídas**: Registra o que você ganha e o que você gasta.
- **Extrato**: Mostra a lista de tudo o que foi anotado.
- **Por categoria**: Organiza os gastos por tipo de despesa.
- **Relatório**: Salva um resumo das contas em um arquivo de texto.

# Teclonogias usadas
- Python
- Json

  # Como utilizar o sistema
  - Salve o código em um arquivo chamado app.py.
  - Abra o terminal na mesma pasta do arquivo.
  - Execute o comando python financas.py e aperte Enter.
